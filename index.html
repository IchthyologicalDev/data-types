Data Types Exercise

Description

Welcome to the Data Types Practice Exercise! This exercise covers strings, numbers, booleans, and bigints. There are many other data types in JavaScript that you'll encounter, but these 4 are the most common when describing basic information.

Instructions

  1. Open up index.js in your editor
  2. Follow the instructions in the comment at the top of that file
  3. Refresh this page after saving changes to see your progress
